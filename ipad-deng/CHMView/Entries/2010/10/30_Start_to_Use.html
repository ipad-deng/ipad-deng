CHMView

 
 
 
 
 

CHMView is a compiled HTML(.chm) file reader for iPad.


FEATURES:

View CHM files with ease,

Automatic open latest file,

Customizable text size,

Reading by scrolling,

Navigate through Table of Contents,

Bookmark Management with ease,

iTunes File Sharing.


You can use it from following step

  1. 1.Add CHM files through iTunes File Sharing.

  2. 2.Open the CHM files.

  3. 3.Select index from popover view or left view.

  4. 4.View the contents.

  5. 5.Add bookmarks.
